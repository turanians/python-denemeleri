# Physics-Based Simulations in Python

Bu depo, temel fizik yasalarını ve kuantum mekaniği prensiplerini Python kullanarak modelleme ve simüle etme çalışmalarımdan oluşmaktadır. Amacım, karmaşık fiziksel fenomenleri sayısal yöntemlerle görselleştirmek ve Python'ın veri analizi/bilimsel hesaplama kütüphanelerini kullanarak çözümler üretmektir.

## 🔬 Çalışma Alanları
- **Kuantum Mekaniği:** Schrödinger denklemlerinin nümerik çözümleri ve dalga fonksiyonu simülasyonları.
- **Dalga Optiği:** Girişim (interference) ve kırınım (diffraction) modelleri.
- **Sayısal Analiz:** Fizik problemlerini çözmek için kullanılan algoritmalar ve optimizasyon yaklaşımları.
